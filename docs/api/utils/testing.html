jobcontrol.utils.testing.testing_job(steps=10, sleep=1, retval='DONE', fail=False)[source]¶

Job to be used for testing purposes.

Provides facilities for simulating various execution scenarios, such as logging, failures, progress, ..

Parameters:
  • steps – How many “steps” this job is composed of
  • sleep – How many seconds to sleep between each “step”
  • retval – What to return
  • fail –
    • if False, the build will succeed
    • if True, the build will fail
    • if an integer, the step at wich the build will fail
    • if a float (0 <= x <= 1), the chance of the job failing
